# Movie Details View Project
###### This project is for viewing movie details like Release Year, Movie Name, Poster , Movie Rating etc.

## Tech Stack
#### Reactjs
#### Firebase Hosting
#### IMDB API
#### redux/toolkit and react-redux
#### react-router-dom

# Getting Started

###### Clone this project from git repo and open project in IDE like Visual Studio
###### Run command "npm i or npm install" to install all dependencies
###### To run project in local type npm start in the IDE Terminal
